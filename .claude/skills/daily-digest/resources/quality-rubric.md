# Quality Rubric

Score each insight candidate on four dimensions:

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| Novelty | Known / obvious | Somewhat new | New + non-obvious |
| Evidence | None | Mentioned | Direct quote |
| Specificity | Generic | Somewhat | Concrete |
| Actionability | Observation | Implies action | Clear next step |

**Inclusion rule**: Include only insights that score 2 on ≥ 3 dimensions. No padding.

## Final Content Counts

| Section | Min | Max |
|---|---|---|
| Key Insights | 1 | 3 |
| Anti-patterns | 2 | 4 |
| Actions to Try | 1 | 3 |
| Resources | 3 | 5 |

If any section falls below its minimum, add the quality warning at the bottom of the digest.
